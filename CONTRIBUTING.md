# Contributing

Thanks for your interest!

This repository is currently a **pre-release skeleton**. There is no working implementation yet — only scaffolding and design notes. Expect the code surface, skill boundaries, and APIs to be rewritten before any 0.1.x tag.

## What you can do right now

- Open an issue to discuss the design (see [DESIGN.md](DESIGN.md)).
- Suggest additional paper sources, export formats, or skill ideas.
- Flag open questions listed in the design document.

## What to hold off on

- Pull requests that add skill implementations. The skill surface is not yet frozen, and contributions made now will likely be rewritten.
- Packaging / distribution changes until the first tagged release.

Once an MVP lands, this document will be replaced with full contribution guidelines.
